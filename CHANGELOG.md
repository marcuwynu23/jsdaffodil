## [v2.0.1](https://github.com/marcuwynu23/jsdaffodil/compare/v2.0.0...v2.0.1) - 2026-03-16

### Added

- Added `watch()`-based CI/CD examples for both ESM and CommonJS in `samples/watch-sample.*`.
- Added multi-host deployment examples using `inventory.ini` in `samples/inventory-sample.*`.
- Added `inventory.ini.example` to document inventory file format.

### Changed

- Updated `README.md` to document `watch()` API, multi-host deployments, and new configuration options (`inventory`, `group`, `verbose`).
- Improved multi-host deployment behavior to reuse the same `Daffodil` instance per host, ensuring `sshCommand` calls in user steps work correctly.
- Relaxed `pre-push` Husky hooks to verify `CHANGELOG.md` instead of regenerating it on every tag push.
